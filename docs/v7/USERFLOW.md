```

  ├── NO  → [① 온보딩 Why Tree 7문항]

  [③ 트렌드]        [④ 라이벌]
       │                 │
       └────────┬────────┘
                ▼
        [⑤ 제작 — 스크립트 + 콘티]
```

## ① 온보딩 — LLM 대화

> Claude API와 자연스러운 대화로 크리에이터의 동기·강점·타깃을 끌어냄  
> 선택지가 아니라 유저가 직접 입력(자유 서술)하는 채팅 형식

```
[Welcome]
  "당신의 첫 영상을 함께 만들어요"
  깔끔한 화이트 배경, 챌린저스 앱 톤
      │
      ▼
[LLM 대화]
  AI: "왜 숏폼을 시작하고 싶으세요?"
   →  유저 답변
  AI: (답변에 따라) "어떤 걸 만들 때 제일 즐거우세요?"
   →  유저 답변
  AI: "주변에서 뭘 잘한다고 하세요?"
   →  ...
  AI: "누가 봐주면 좋겠어요?"
   →  ...
  * 동기 → 강점 → 즐거움 → 타깃 흐름으로 자연스럽게,
    답변에 따라 후속 질문이 달라짐
      │
      ▼
[Loading]
  POST /api/persona (Claude) → 대화 분석
      │
      ▼
[② 대화 결과 카드]
```

**localStorage 저장:** onboardingDone, 대화 로그, personaResult

## ② 설문 결과 — "그럼 뭐부터 찍어볼까?"

> 설문 결과를 한눈에 정리 + 첫 제작 유도

```
[결과 카드]
  ┌─────────────────────────────┐
  │ 당신은 이런 크리에이터예요   │
  │                             │
  │ 콘텐츠 방향: [한 줄 요약]    │
  │ 강점: [키워드]               │
  │ 추천 타깃: [누구]            │
  │ 추천 포맷: [어떤 형식]       │
  └─────────────────────────────┘
      │
      ▼
  "그럼 뭐부터 찍어볼까요?"
  [트렌드 보러가기 →]
```

## ③ 트렌드 레퍼런스

> 관련 키워드 조합으로 지금 뜨는 콘텐츠 보여줌

```
[트렌드 화면]
  헤더: "[카테고리] 지금 뜨는 영상"
  │
  ├─ 지금 뜨는 영상 리스트
  │   각 카드: 제목 · 조회수 · ER% · 왜 떴는지 한 줄
  │   GET /api/trends + POST /api/insights
  │
  └─ 트렌드 카드 탭
      → "이 주제로 만들어보기 →" → [⑤ 제작]
```

> ⚠️ 데이터 한계: 영상 내용 분석 불가, 텍스트 메타데이터 기반  
> 포지셔닝 = "분석"이 아니라 "지금 뜨는 레퍼런스"

## ④ 라이벌 크리에이터

> 나와 비슷한 주제 · 비슷한 수준의 크리에이터 추천

```
[라이벌 화면]
  "당신과 비슷한 크리에이터"
  │
  └─ 크리에이터 카드
      핸들 · 팔로워 · 주제 · 최근 성장
      ⚠️ "약간 앞선 수준"만 (위축 방지)
      "이 사람도 3개월 전엔 0이었어요" 서사
```

## ⑤ 제작 — 스크립트 + 콘티

> 트렌드 주제를 실제로 만들 수 있게

```
[제작 화면]
  선택한 트렌드 배너
      │
      ▼
  ┌──────────────┬──────────────┐
  │ 스크립트 받기 │  콘티 받기    │
  └──────┬───────┴──────┬───────┘
         ▼              ▼
  POST /api/generate  POST /api/conti
  대본 3종            콘티 4컷
  (정보형/스토리/훅)   (훅/전환/본론/클로징)
         │              │
         └──────┬───────┘
                ▼
       [저장하기] → localStorage
       [다음 주제 추천받기] → ③으로 루프
```

## API 매핑

| 시점 | 엔드포인트 | 모델 | 캐싱 |
|------|-----------|------|------|
