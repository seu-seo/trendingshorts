# WHYTREE — Shortform Pulse

> Why Tree 분석 결과  
> v7 · 2026-06-13

---

## 문제 정의

> 초기 크리에이터가 숏폼을 시작하지 못하거나, 시작해도 금방 포기한다.

---

## Why Up — 왜 이 문제를 풀어야 하는가

```
초기 크리에이터가 포기한다
    │
    ├─ WHY? 알고리즘의 선택을 받지 못해서
    │       └─ WHY? 뭘 어떻게 찍어야 터지는지 모르니까
    │               └─ WHY? 크리에이터 생태계를 모른 채 시작하니까
    │
    ├─ WHY? 콘텐츠 제작 동력을 잃어서
    │       └─ WHY? 반응(조회수·댓글)이 없으니까
    │               └─ WHY? 첫 N개를 버티지 못하니까
    │
    └─ 결과: 크리에이터 생태계에 신규 진입자가 안착하지 못함
            → 다양성 감소, 소수 대형 크리에이터 독점 심화
```

**핵심:** 진입장벽이 높아서가 아니라, **진입 후 첫 성공까지의 공백**이 너무 길어서 포기한다.

---

## Why Down — 왜 이런 일이 일어나는가 (근본 원인)

```
초보 크리에이터가 막막하다
    │
    ├─ 원인 1. 정보 비대칭
    │   └─ 개인화 알고리즘은 내 소비 패턴만 보여줌
    │       └─ 대중이 지금 뭘 보는지(트렌드)를 알 수 없음
    │           └─ 해결: 내 카테고리의 지금 트렌드를 보여준다
    │
    ├─ 원인 2. 자기 정체성 부재
    │   └─ "뭘 찍고 싶은지" 스스로도 모름
    │       └─ 막연히 "유명해지고 싶다"만 있음
    │           └─ 해결: Why Tree 온보딩으로 동기·강점·타깃을 끌어냄
    │
    ├─ 원인 3. 실행 격차
    │   └─ 아이디어가 있어도 "어떻게 찍지?"에서 막힘
    │       └─ 해결: 스크립트 + 콘티로 바로 찍을 수 있게
    │
    └─ 원인 4. 고립감
        └─ 나만 안 되는 것 같음, 비교 대상이 없음
            └─ 해결: 비슷한 수준의 라이벌 크리에이터를 보여줌
```

---

## 해결책 매핑

| 근본 원인 | 우리 기능 | 효과 |
|----------|----------|------|
| 정보 비대칭 | 트렌드 레퍼런스 추천 | 지금 뭐가 뜨는지 안다 |
| 자기 정체성 부재 | Why Tree 온보딩 7문항 | 내가 뭘 해야 할지 안다 |
| 실행 격차 | 스크립트 + 콘티 제안 | 바로 찍을 수 있다 |
| 고립감 | 라이벌 크리에이터 추천 | 나도 할 수 있다고 느낀다 |

---

## 검증해야 할 가정

- [ ] 초보 크리에이터가 대화에서 "왜 찍고 싶은지"를 자연스럽게 답할 수 있는가 (너무 어렵지 않은가)
- [ ] Why Tree 결과가 실제로 "아하" 모먼트를 주는가
- [ ] 라이벌 크리에이터 추천이 동기부여가 되는가, 오히려 위축시키는가
- [ ] 텍스트 기반 트렌드 추천이 신뢰할 만한가
- [ ] 콘티를 보고 실제로 따라 찍는가
