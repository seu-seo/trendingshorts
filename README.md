# Shortform Pulse

크리에이터를 위한 올인원 숏폼 트렌드 인텔리전스 플랫폼

## Overview

Shortform Pulse는 YouTube Shorts, TikTok, Instagram Reels의 트렌딩 데이터를 실시간으로 수집하고, 크리에이터와 마케터가 데이터 기반으로 콘텐츠 의사결정을 할 수 있도록 돕는 플랫폼입니다.

## Core Features

- **Real-time Trending Feed** — 3개 플랫폼 트렌딩 피드 통합
- **Cross-Platform Compare** — 동일 카테고리 플랫폼별 성과 비교
- **Creator Insights** — 최적 업로드 시간, 영상 길이, 포맷 데이터
- **Trend Alerts** — 관심 카테고리 바이럴 감지 즉시 알림

## Pilot Case

에이카랩스 (D2C 칫솔/치약 브랜드)와 함께 트렌드 기반 마케팅 소재 제작 및 성과 검증을 진행합니다.
