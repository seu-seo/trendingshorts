# Pulse — Product Spec

> 한국 크리에이터를 위한 AI 트렌드 인사이트 앱  
> 버전: v6 (프리미엄 다크 / Kraster-style)  
> 최종 업데이트: 2026-06-01

---

## 1. 제품 개요

### 한 줄 정의
트렌드 데이터에 **WHY**를 더하는 크리에이터 인사이트 앱.

### 핵심 차별점
| 기존 트렌드 앱 | Pulse |
|---|---|
| 숫자/순위만 보여줌 | 숫자 + 왜 뜨는지 설명 |
| 정보 소비로 끝 | 콘텐츠 실행까지 연결 |
| 일반 사용자 대상 | 한국 크리에이터 특화 |

### 타겟
- 인스타그램 · 틱톡 · 유튜브 운영하는 한국 크리에이터
- 팔로워 1,000~50,000 구간 (성장 단계)
- 매일 "뭘 올려야 하지?" 고민하는 사람

Loading

## 3. 주요 화면별 기능

### 3-1. 트렌드 대시보드

| 섹션 | 내용 |
|---|---|
| Today's Big Trend | 헤드라인 트렌드 1개 + WHY 인사이트 카드 |
| 실시간 키워드 | 버블 형태 키워드 5개 (크기 = 성장률) |
| 지금 뜨고 있어요 | Top 5 트렌드 리스트 (저장 버튼 포함) |
| Live Pulse Mode | 어제 본 트렌드 추적 카드 (가로 스크롤) |

**3색 데이터 시스템**
- 🔴 `#FF4274` — 성장/상승 (뜨거운 트렌드)
- 🔵 `#38B6FF` — 하락/식는 중
- 🟢 `#10B981` — 중립/플랫폼 태그

### 3-2. 마이페이지

| 섹션 | 내용 |
|---|---|
| 프로필 카드 | 닉네임 · 팔로워 · 플랫폼 태그 |
| This Week's Insights | 주간 성과 지표 (4개 KPI 카드) |
| 01 콘텐츠 만들기 | AI 니치 분석 + 대본/콘티 추천 |
| 02 목표 설정하기 | KPI 설정 |
| 03 계정 연결하기 | 채널 데이터 연동 |
| 내 컬렉션 | 저장한 트렌드 그리드 |
| 저장한 대본·콘티 | 저장한 대본/콘티 목록 |

### 3-3. 콘텐츠 만들기 플로우

```
마이페이지 "콘텐츠 만들기" 클릭
  ↓
[질문 폼] — 한 화면에 7개 질문 동시 표시
  열정 · PASSION   — 1년 내내 만들고 싶은 것
  소비 · SPENDING  — 가장 돈 많이 쓰는 취미
  강점 · STRENGTH  — 주변에서 칭찬받는 것
  오디언스 · AUDIENCE — 보여주고 싶은 사람
  주제 · TOPIC     — 이번 영상 주제
  포맷 · FORMAT    — 형식/촬영 환경
  훅 · HOOK        — 첫 3초 전략
  ↓
[AI 분석 시작] 버튼
  ↓
[AI 니치 분석 완료] 결과 카드
  - 니치 타이틀
  - 설명 텍스트
  - 키워드 태그
  - FORMAT · PLATFORM 추천
  ↓
┌──────────────┬──────────────┐
│ 대본 추천 받기 │ 콘티 추천 받기 │
└──────────────┴──────────────┘
  ↓
[주제 입력] — 어떤 영상을 만들고 싶어요?
  ↓
[대본 결과] or [콘티 결과]
  - 이 대본/콘티 저장하기
  - 마이페이지로 돌아가기
```

**니치 분류 (키워드 기반 AI 분석)**
| 니치 | 포맷 | 플랫폼 |
|---|---|---|
| 집밥 & 카페 큐레이터 | 쇼츠 · 30-60초 원테이크 | 인스타그램 릴스 |
| 현실 피트니스 크리에이터 | 7-30일 챌린지 기록 | 틱톡 · 유튜브 쇼츠 |
| 로컬 여행 큐레이터 | 당일치기 코스 · 장소 리뷰 | 인스타그램 · 유튜브 |
| AI · 테크 라이프 크리에이터 | "써봤어요" 비교 리뷰 | 유튜브 쇼츠 · 틱톡 |
| 생산성 · 자기계발 크리에이터 | OO일 챌린지 · 루틴 기록 | 유튜브 · 인스타그램 |

## 4. 디자인 시스템

### 컬러 토큰
```css
--bg:           #0A0A0A   /* 메인 배경 — 순수 블랙 */
--bg-soft:      #111111
--bg-card:      #1A1A1A   /* 카드 표면 */
--bg-hover:     #222222

--primary:      #C8FF57   /* Acid Lime — 메인 액센트 */
--up:           #FF4274   /* 핑크-레드 — 성장 수치 */
--down:         #38B6FF   /* 일렉트릭 블루 — 하락 */
--neutral:      #10B981   /* 테크 민트 — 중립 */

--ink:          #FFFFFF   /* 메인 텍스트 */
--ink-soft:     #A1A1AA   /* 서브 텍스트 */
--line:         #2A2A2A
--line-strong:  #333333
```

### 타이포그래피
| 역할 | 폰트 | 용도 |
|---|---|---|
| 임팩트 헤딩 | 카페24 당당해 | 숫자 강조, 브랜드 포인트 |
| 친근한 헤딩 | 카페24 써라운드 | 질문 텍스트 |
| 본문 | Pretendard | 설명, 보조 텍스트 |
| 데이터/레이블 | Inter | 수치, 영문 메타, uppercase 레이블 |

### 디자인 원칙
1. **이모지·이미지 금지** — 타이포/색/도형만으로 위계 표현
2. **그라디언트 없음** — flat + 강한 타이포로 승부
3. **액센트는 아껴서** — CTA 버튼, 핵심 수치에만
4. **WHY 인사이트** — 숫자만 보여주면 차트 앱, WHY가 있어야 인사이트 앱

## 5. 기술 스펙

### 현재 구조
- **단일 파일**: `demo/v6/index.html` (CSS + JS 인라인)
- **상태 관리**: in-memory JS 변수 (새로고침 시 리셋)
- **데이터**: 하드코딩 mock 데이터
- **화면 전환**: `.screen` + `.active` class toggle (`goTo()` 함수)
- **탭 전환**: `switchTab('trend' | 'my')`

### 주요 JS 함수
| 함수 | 역할 |
|---|---|
| `skipToSampleDashboard()` | 온보딩 스킵, 대시보드 진입 |
| `openDeepProfile()` | 콘텐츠 만들기 폼 열기 |
| `submitDeepProfile()` | 7개 답변 수집 → AI 분석 결과 표시 |
| `buildDeepResult()` | 키워드 분류 → 니치 카드 렌더링 |
| `openScript() / openStoryboard()` | 주제 입력 화면 → 대본/콘티로 이동 |
| `saveContent(type)` | 대본/콘티 저장 |
| `saveTrend(...)` | 트렌드 저장 |
| `renderCollection()` | 내 컬렉션 렌더링 |
| `renderSavedContents()` | 저장한 대본·콘티 렌더링 |
