# 프로젝트 방향 (Project Approach)

## 1) What to Build
- 숏폼 트렌드 분석 → 마케팅 소재화까지 가능하게 해주는 AI 툴

## 2) How to Implement
- 유사한 고민을 갖고 있는 기업에 실제 적용
  - 신규 유입이 '프로모션 의존적'인 에이카랩스
  - 오가닉 or paid 소재를 트렌드에 맞게 제작했을 때 관심도 및 매출이 늘어나는지 확인해보는 것이 목표

## 3) Further Approach
- 마케팅 및 매출 성과까지 리포트할 수 있는 대시보드까지 연결
  - 에이카랩스의 문제 중 '통합 데이터 파이프라인 부재 / 광고-오가닉 상관관계' 등과 연관
