# MANIFEST.md
### Shortform Pulse — 크리에이터를 위한 올인원 숏폼 트렌드 인텔리전스 플랫폼

---

## 왜 만드는가

숏폼 크리에이터가 가장 먼저 부딪히는 문제는 아이디어가 없는 게 아니다. **지금 대중이 무엇을 보는지 알 방법이 없다는 것이다.**

여기에 구조적 역설이 있다. 소비자용 알고리즘은 정교하게 개인화되어 있다. 그런데 그 개인화가 크리에이터에게는 독이 된다. 내 피드는 내 소비 패턴만 반영한다. 크리에이터가 가장 필요로 하는 데이터 — 내 버블 밖에서 대중이 무엇을 보는가 — 를, 플랫폼 알고리즘은 가장 노출하지 않는다.

```
소비자 모드                         크리에이터 모드
─────────────                      ────────────────
내 선호 콘텐츠                        대중 선호 콘텐츠
     ↓                                     ↓
알고리즘이 최적화해서 노출             알고리즘이 노출하지 않음
     ↓                                     ↓
정보 과잉                              정보 비대칭 발생
```

이 정보 비대칭은 자체 데이터팀이나 엔터프라이즈 툴을 갖춘 기성 크리에이터(팔로워 10만+)는 자체 해결할 수 있다. **팔로워 1만 명 미만의 초기 크리에이터에게는 해소 수단이 없다.** 결과는 두 개의 숫자로 나온다. YouTube 채널의 90% 이상이 구독자 1만 명에 도달하지 못하고, 크리에이터의 90%가 번아웃을 경험한다.

기존 툴(VidIQ, TubeBuddy 등)은 이 문제의 절반만 해결한다. *"무엇이 뜨는가"* 까지는 알려준다. 그런데 크리에이터에게 실제로 필요한 것은 *"그래서 나는 오늘 뭘 찍어야 하는가"* 다. 기존 툴은 거기서 멈춘다.

**Shortform Pulse는 그 다음까지 간다.** 트렌드 발견에서 대본 초안 생성까지, 크리에이터의 의사결정 워크플로우 전체를 하나의 흐름으로 완성한다.

## 타겟

**팔로워 1만 명 미만 초기 크리에이터**

- 콘텐츠 제작 경력 3년 미만
- 월 0~10편 산출
- 부업 또는 풀타임 전환 초기
- 자체 데이터 인프라를 구축할 예산과 시간이 없다
- 1만 명은 광고 수익 진입선이자 데이터 인프라 구축 자원의 임계점

**부가 파일럿: 에이카랩스 (스타트업 마케터)**
신규 유입이 프로모션에 의존적인 에이카랩스가 트렌드 기반 비가격 소재를 만들었을 때 성과가 올라가는지 검증하는 케이스로 활용한다. 크리에이터 타겟과 별개의 세그먼트로 관리한다.

## 제품 정의

```
대중이 무엇을 보는지 알 수 없다는 정보 비대칭을 해소하고 →
3개 플랫폼의 트렌딩 데이터를 서브카테고리 단위로 가공해서 →
"오늘 뭘 찍을지" 결정부터 대본 초안까지 한 흐름으로 완성한다
```

## 경쟁 툴과의 차이

| | VidIQ / TubeBuddy | Exploding Topics | **Shortform Pulse** |
|--|--|--|--|
| 플랫폼 | 단일 (YouTube 중심) | 웹 트렌드 중심 | YouTube + TikTok + Instagram |
| 추천 단위 | 카테고리 | 토픽 | **서브카테고리** (플랫폼별 분리) |
| 워크플로우 종착점 | 트렌드 발견 | 트렌드 발견 | **대본 초안 생성까지** |
| 타겟 | 중대형 채널 중심 | 마케터·투자자 | 팔로워 1만 이하 초기 크리에이터 |
| 가격 | 월 $10~$50+ | 월 $39+ | 초기 크리에이터 접근 가능 수준 |

기존 툴은 *"무엇이 뜨는가"* 까지만 알려준다. Shortform Pulse는 *"그래서 너는 오늘 뭘 찍어야 하는가"* 까지 워크플로우를 완성한다.

서브카테고리 단위 추천이 핵심 차별점이다. "먹방"이라는 카테고리 안에서도 혼밥 ASMR, 편의점 리뷰, 레시피 챌린지는 플랫폼별로 성과가 전혀 다르다. 카테고리 단위 추천은 이 차이를 무시한다. Shortform Pulse는 Claude 기반 서브카테고리 자동 분류기로 이 레벨까지 내려간다.

## 핵심 기능

### 📊 Real-time Trending Feed (대시보드 탭 — 첫 화면)
YouTube, TikTok, Instagram의 트렌딩 숏폼을 실시간으로 수집한다. **서브카테고리 단위** 필터링이 핵심이다. "먹방"이 아닌 "편의점 리뷰"처럼, 실제로 지금 뜨는 콘텐츠 유형을 플랫폼별로 구분해서 보여준다.

### ⭐ For You (추천 탭)
페르소나 기반 맞춤 트렌드를 추천한다. 설정 전에도 기본 추천이 제공되고, 주 카테고리와 콘텐츠 스타일을 설정하면 정확도가 올라간다. 강제 온보딩이 아닌 자발적 설정 구조다.

### ✏️ 대본 초안 생성 (제작 탭)
발견한 트렌드를 바탕으로 3-part 대본 초안을 자동 생성한다.
- **HOOK** — 첫 3초, 클릭을 유도하는 오프닝
- **BODY** — 3단 내러티브 구조
- **CTA** — 구독·좋아요·댓글 유도

기존 툴에서 트렌드를 발견한 뒤 크리에이터가 스스로 해야 했던 "그래서 뭘 만들지"를 처음으로 자동화한다.

## 핵심 가치

### 1. 정보 비대칭을 해소한다
내 피드는 내 버블만 보여준다. 크리에이터에게 필요한 데이터(대중이 무엇을 보는가)를 플랫폼 알고리즘은 노출하지 않는다. Shortform Pulse는 이 역설을 깨고 대중의 집계된 행동 데이터를 크리에이터 관점으로 재가공한다.

### 2. 카테고리가 아닌 서브카테고리로 추천한다
"먹방을 찍으세요"는 쓸모없는 추천이다. "이번 주 TikTok에서 가장 빠르게 성장하는 건 편의점 리뷰고, YouTube Shorts에서는 레시피 챌린지다" — 이게 실제로 쓸 수 있는 인사이트다. Claude 기반 서브카테고리 자동 분류기가 이걸 가능하게 한다.

### 3. "무엇이 뜨는가"에서 "오늘 뭘 찍을지"까지 완성한다
기존 툴이 멈추는 지점이 우리의 시작점이다. 트렌드 발견(대시보드) → 맞춤 추천(For You) → 대본 초안(제작 탭)이 하나의 흐름으로 이어진다. 크리에이터가 툴을 닫고 다른 곳으로 갈 필요가 없다.

### 4. 시간 단축이 곧 경제적 가치다
크리에이터가 "오늘 찍을 것"을 정하는 데 현재 30~90분이 걸린다. 한국 숏폼 편집 단가의 53.9%가 1만~1.5만 원 구간이다(커넥스페이스, 2025). 리서치 시간을 줄이는 것 자체가 측정 가능한 경제적 가치다.

### 5. 초기 크리에이터 수준에서 접근 가능하다
VidIQ, TubeBuddy는 예산과 여유 시간이 있는 중대형 채널을 전제로 한다. Shortform Pulse는 팔로워 1만 이하, 아직 수익이 없는 크리에이터를 전제로 설계됐다.

## 설계 원칙

| 원칙 | 이유 |
|------|------|
| 정보 비대칭 해소가 첫 번째 목적 | 플랫폼 알고리즘이 크리에이터에게 역행하는 구조를 뒤집는 것이 핵심 |
| 서브카테고리 단위가 기본값 | 카테고리 추천은 기존 툴이 이미 한다. 차별점은 그 안에 있다 |
| 대본까지 한 흐름 | 트렌드 발견에서 멈추면 크리에이터는 또 다른 툴을 열어야 한다 |
| 가치를 먼저, 설정은 나중에 | 강제 온보딩보다 제품을 먼저 보여주고 필요를 느낀 사용자가 설정한다 |
| 실시간 수집이 기본 | 24~48시간 내에 교체되는 숏폼 트렌드에서 어제 데이터는 이미 지났다 |
| 데이터에는 출처가 있어야 한다 | 숫자만 있고 근거가 없으면 의사결정을 신뢰할 수 없다 |

## 우리가 만들지 않는 것

- 각 플랫폼이 이미 제공하는 단순 인기 영상 목록
- 트렌드와 무관하게 AI가 만들어주는 범용 콘텐츠 아이디어
- 카테고리 수준에서 멈추는 트렌드 리포트
- 사후 분석만 되는 리포팅 툴 (실시간성 없음)
- 팔로워 10만+ 또는 에이전시를 전제로 한 엔터프라이즈 대시보드
- 강제 설문 없이는 아무것도 볼 수 없는 온보딩 구조
