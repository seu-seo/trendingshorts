# MANIFEST.md
### Shortform Pulse — 크리에이터를 위한 올인원 숏폼 트렌드 인텔리전스 플랫폼

---

## 왜 만드는가

숏폼 콘텐츠를 시작하는 사람들이 가장 먼저 부딪히는 문제는 아이디어가 없는 게 아니다. 지금 무엇이 뜨고 있는지 알 방법이 없다는 것이다. YouTube Shorts, TikTok, Instagram Reels — 각 플랫폼을 따로따로 돌아다니며 수동으로 확인하다 보면 이미 트렌드는 지나가 있다. 숏폼 트렌드는 24~48시간 내에 교체된다. 수동 리서치로는 구조적으로 따라갈 수 없는 속도다.

결과는 두 개의 숫자로 나온다. YouTube 채널의 90% 이상이 구독자 1만 명에 도달하지 못하고, 크리에이터의 90%가 번아웃을 경험한다. 만들고 올렸는데 아무도 보지 않는 경험이 반복되면 지친다. 그 원인 중 핵심이 트렌드 파악 실패다.

Shortform Pulse는 이 문제를 해결한다. 3개 플랫폼의 트렌딩 데이터를 하나의 대시보드에서 보여주고, 크리에이터와 마케터가 감이 아닌 데이터로 콘텐츠를 결정할 수 있게 한다.

## 제품 정의

```
3개 플랫폼의 트렌딩 데이터를 실시간으로 수집하고 →
크리에이터와 마케터에게 의미 있는 인사이트로 가공해서 →
콘텐츠 의사결정을 감이 아닌 데이터로 만든다
```

**파일럿 케이스:** 에이카랩스 (어린이·성인 치약·칫솔 D2C 브랜드)
신규 유입이 프로모션에 의존적인 에이카랩스가 트렌드 기반 비가격 소재를 만들었을 때 성과가 올라가는지 검증하는 케이스로 활용한다.

## 핵심 기능

### �� Real-time Trending Feed
YouTube, TikTok, Instagram의 트렌딩 숏폼을 실시간으로 수집한다. 카테고리, 해시태그, 성장률 기준으로 정렬하고 필터링할 수 있다. 플랫폼을 따로 돌아다닐 필요가 없다.

### ⚡ Cross-Platform Compare
같은 카테고리 콘텐츠가 어느 플랫폼에서 더 잘 되는지 나란히 비교한다. 각 플랫폼 평균 인게이지먼트(YouTube 5.91%, TikTok 3.70%, Instagram 1.23%)를 기준점으로 성과를 판단한다.

### �� Creator Insights
최적 업로드 시간, 효과적인 영상 길이, 인기 포맷 패턴을 데이터로 보여준다. 크리에이터가 감이 아닌 숫자로 의사결정할 수 있게 한다.

### �� Trend Alerts
관심 카테고리에서 새로운 바이럴이 감지되면 즉시 알린다. 트렌드를 따라가는 게 아니라 선점하는 것이 목적이다.

## 핵심 가치

### 1. 분산된 플랫폼을 하나로 본다
크리에이터가 3개 플랫폼을 따로 확인하는 데 쓰는 시간을 없앤다. 모든 트렌딩 데이터가 한 화면에 있어야 의미 있는 비교가 된다.

### 2. 감이 아닌 데이터로 결정한다
"이게 뜰 것 같다"는 판단을 "이 카테고리는 현재 성장률 X%, 플랫폼 평균 대비 인게이지먼트 Y배"로 바꾼다.
플랫폼별 평균 인게이지먼트(YouTube Shorts 5.91%, TikTok 3.70%, Instagram Reels 1.23%, Social Insider 2025-2026)를 기준점으로 삼아, 
어떤 플랫폼에서 어떤 카테고리가 실제로 성과를 내는지 데이터로 판단한다.

### 3. 속도가 핵심이다
숏폼 트렌드는 24~48시간 내에 교체된다. 발견이 늦으면 이미 의미가 없다. 실시간 수집과 즉각 알림이 이 문제를 해결한다.

### 4. 크리에이터와 마케터 모두를 위한다
개인 크리에이터는 트렌드 파악과 인사이트를 원하고, 스타트업 마케터는 트렌드를 브랜드 소재로 연결하는 것까지 원한다. 두 사용 패턴을 모두 커버한다.

## 설계 원칙

| 원칙 | 이유 |
|------|------|
| 실시간 수집이 기본 | 캐시된 어제 데이터는 이미 지난 트렌드다 |
| 크로스플랫폼 비교를 핵심에 놓는다 | 단일 플랫폼 분석은 각 플랫폼 자체에서 제공한다 |
| 알림은 선점을 위한 것 | 이미 뜬 걸 아는 건 늦다 |
| 데이터에는 출처가 있어야 한다 | 숫자만 있고 근거가 없으면 의사결정을 신뢰할 수 없다 |
| 마케터 use case는 레이어로 추가 | 트렌드 발견이 기반이고 소재화는 그 위에 얹힌다 |

## 우리가 만들지 않는 것

- 각 플랫폼이 이미 제공하는 단순 인기 영상 목록
- AI가 만들어주는 범용 콘텐츠 아이디어 (브랜드 맥락 없음)
- 사후 분석만 되는 리포팅 툴 (실시간성 없음)
- 개발자만 설정할 수 있는 복잡한 API 대시보드
